# API를 이용한 TimesNews 

- ##  API 제공사이트 : https://newsapi.org/

####  - cors
####  - axios , express
####  - 부트스트랩 DatePicker 사용 (날짜선택기능추가)

- ## 기능

### 1. 첫페이지 로딩시, 최근 뉴스 보이기
### 2. keyword 와 date 선택시 해당하는 뉴스가 렌더링된다.
### 3. 페이지 네이션 구현: 한 페이지당 10개 컨텐츠. 그룹사이즈 5.
### 4. 현재 local server에서만 실행됨

- ## 추가구현할 사항
####  뉴스 컨텐츠 클릭시, 세부뉴스 렌더링되게 하기.
